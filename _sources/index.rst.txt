=================================================
|product-name|
=================================================

|product-name| is a cross-platform library aimed at system integrators willing to quickly add state-of-the-art,
standardized and sensor independent fingerprint recognition technology to their products. 

It is available as a Software Development Kit (SDK) offering a comprehensive interface to simplify integration of the library on servers, desktops/laptops, mobile and edge devices.

This document is the developer's guide of |product-name|.
