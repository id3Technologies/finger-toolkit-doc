Adding the library to your project
==================================

.. tab-set::

    .. tab-item:: Python
        :sync: Python

        .. code-block:: python

            import id3finger

    .. tab-item:: Dart
        :sync: Dart

        .. code-block:: dart

            import 'package:id3_finger/id3_finger.dart';

    .. tab-item:: C#
        :sync: C#

        id3 Finger C# API is available in the package `id3.Finger`:

        .. code-block:: csharp

            using id3.Finger;

    .. tab-item:: Java
        :sync: Java
   
        .. code-block:: java
        
            import eu.id3.finger;

    .. tab-item:: C
        :sync: C

        .. code-block:: c

            #include "id3Finger.h"
